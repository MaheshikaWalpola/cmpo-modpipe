# Code of Conduct

This repository is a research artifact. We follow the spirit of the
Contributor Covenant (v2.1): be respectful and constructive in issues and
pull requests, assume good faith, and focus on the work. Report problems
with the artifact via GitHub issues; reproduction reports (including
failures) are especially welcome and will be acknowledged. Contact the
corresponding author (see CITATION.cff) for anything that should not be
public.
